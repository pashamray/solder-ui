# Install

eim - https://docs.espressif.com/projects/esp-idf/en/stable/esp32/get-started/index.html#installation

install dependencies

```shell
$ ~/.espressif/v6.0.1/esp-idf/install.sh
```

or


```shell
$ ~/.espressif/v6.0.1/esp-idf/install.fish
```

export in project


```shell
$ . ~/.espressif/v6.0.1/esp-idf/export.sh
```

or 

```shell
$ . ~/.espressif/v6.0.1/esp-idf/export.fish
```

# Build and flash

```shell
$ idf.py build
```

```shell
$ idf.py flash
```
